Курс: 1С-Программист: Для начинающих 

Программа курса:

1. Вводный урок в 1С-разработку:
- знакомство с платформой 1С 

2. Базовый интерфейс, хранение данных:
- применение констант, справочников и документов
- cоздание структуры для хранения данных
- настройка удобного пользовательского интерфейса
- создание объектов в базе

3. Основы интеграции с внешними источниками:
- работа с внешними сервисами
- загрузка файлов из Интернета
- создание регистра сведений
- реализация заполнения регистра сведений денежными средствами в указанной валюте

4. Отчёты и дашборды
- создание отчётов и дашбордов для анализа бизнес-процессов
- проведение документов по регистрам накопления
- создание набора отчётов

5. Итоговый тест:
- Выполнение итогового теста на пройденный материал
